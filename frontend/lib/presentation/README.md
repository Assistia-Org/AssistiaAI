# Presentation Layer
Bu katman uygulamanın kullanıcı arayüzünü ve durum yönetimini (state management) barındırır.

## Alt Klasörler
- `pages/` veya `screens/` : Kullanıcıya gösterilen ana ekranlar.
- `widgets/` : Çeşitli sayfalarda tekrar kullanılabilen görsel parçalar.
- `bloc/`, `providers/` veya `controllers/` : State management mantığı ve kullanıcı etkileşimlerini yöneten yapılar.
