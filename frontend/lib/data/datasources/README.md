# DataSources (Veri Kaynakları)
Remote (API / Backend) veya Local (Veri Tabanı, SharedPreferences vb.) verilerinin çağrıldığı veya kaydedildiği yerdir. Sadece ham veri (veri modelleri, JSON vb.) döndürür.

**Örnek Sınıflar:**
- `auth_remote_data_source.dart` (Backend api ile iletişim kurar)
- `user_local_data_source.dart` (Local DB / Cache ile iletişim kurar)
