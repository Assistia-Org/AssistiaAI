# Theme (Tema Ayarları)
Uygulamanın arayüz tasarımını standardize eden Renk paketlerinin (Colors), Font stillerinin, Dark/Light Mode yapılandırmalarının ve app genelinde uygulanacak materyal tasarımın (Material Design) tutulduğu kısımdır.

**Örnek Sınıflar:**
- `app_colors.dart`
- `app_theme.dart`
- `text_styles.dart`
