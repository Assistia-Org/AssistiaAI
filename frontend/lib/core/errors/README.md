# Errors (Hatalar ve İstisnalar)
Data source kaynaklı özel hata (Exception) türleri ile uygulamanın kendi içinde fırlatacağı Failure (Başarısızlık) durumlarının yapıları burada tutulur.

**Örnek Sınıflar:**
- `failures.dart` (ServerFailure, CacheFailure vb.)
- `exceptions.dart` (ServerException, CacheException vb.)
