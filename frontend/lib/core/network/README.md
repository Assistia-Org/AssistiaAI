# Network (Ağ İşlemleri)
Ağ modüllerinin yapılandırıldığı (Dio, Http vb.), interceptor (istek arasına girici) eklemelerinin yapıldığı veya bağlantı durumu kontrol (NetworkInfo) sınıflarının bulunduğu yerdir.

**Örnek Sınıflar:**
- `network_info.dart` (İnternet bağlantısı kontrolü)
- `api_client.dart` / `dio_interceptor.dart`
